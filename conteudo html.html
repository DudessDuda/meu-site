Documento html

Um documento html possui duas partes: Head(cabeçalho), Body(corpo). No head, você entra com o título da página web, que será exibido apenas na barra de endereços do navegador No body, você entra com o conteúdo da página web


Exemplo:


<'html'>
<'head>
<'title'> minha primeira página <'/title>
<'/head>
<'body'>
Hoje o dia está muito ensolarado.
<'/body>
<'/html>

Pra aqueles que opitarem por escrever os códigos em um bloco de notas prescisão saber como salvar o arquivo para ele ser vizualizado no navegador

Salvar:

Para salvar o documento html gerado pelo bloco de notas é preciso colocar a extensão .html ( todo arquivo feito no bloco de notas é salvo com a extensão .txt e ele não é aberto nos navegadores) e colocar a codificação UTF-8( é a recomendação atual para codificação na web, além de ser compatível com quase todos os idiomas) por padrão os documentos criados no bloco de notas são salvos com a decodificação Ansi que é considerada ultrapassada por utilizar um único byte, o UTF-8 utiliza codificação multibyte

< >tags:

Os sinais < > permitem aos navegadores distinguir as tags de um texto comum, o texto pode ser escrito com letra maiúscula e minúscula. A maioria das tags precisam de um fechamento ele é idêntico a tag de abertura, adiciona só o sinal ( / ) no início. (As '' que seram colocadas colocadas dentros das tags serão para que vocês consigam visualizar o código, sendo que eu estou escrevendo isso usam html.)

Exemplos:


<'html'> e <'/html'>: inicia e encerra um documento HTML

<'head'> e <'/head'>: inicia e encerra a área do cabeçalho

<'title'> e <'/title'>: inicia e encerra a área do corpo(conteúdo) do documento html

<'p'> e <'/p'>: inicia e encerra um novo parágrafo

Existem algumas tags que não necessitam de fechamento, por exemplo a tag <'br>

<'br'> e <'/br'>: adiciona uma quebra de linha, utilizasse no final do código já escrito.

Efeitos:


Sublinhado

O efeito sublinhado é pouco utilizado, utilizamos para que os usuários não confundam o texto destacado com links ou quando queremos destacar algum item.

Sintaxe:<'u><'/u>


Exemplo:

<'p>texto da página <'u>web texto da página web


Negrito

O efeito negrito também é utilizado para chamar a atenção do leitor, utilizamos ou quando queremos destacar algum item.

Sintaxe:<'b><'/b>


Exemplo:

<'p>texto da página <'b>web texto da página web


Cor da fonte

Para adicionar a cor ao seu texto você prescisa saber que a duas formas para isso , usando o real nome da cor que você quer atribuir por exemplo:green(ao utilizar o nome da cor ela será restritiva as cores básicas), mas se quiser ter uma variedade maior de cores começaremos por explicar o termo rgba.

RGBA significa red, green, blue e alpha, sendo o corjunto de rgb atribuido as cores primárias (sendo possivel alcançar mais de 16 milhões de variações de cores) e aplpha a opaticidade ou o quão trasparente ou claro você quer deixar algo. Para aqueles que forem opitar de usar o bloco de notas e quiserem usar o rgba basta colocar no google tabela rgba.

Sintaxe:<'spam style="color:"><'/spam>


Exemplo:

<'p>texto da página <'spam style="color:red">web texto da página web


Alterar a fonte e o tamanho da fonte

Para especificar o tamanho da fonte também utilizaremos a sintaxe <'spam style=";"> a alteração do tamanho da fonte utilazaremos o tamnho em pixels. Teoricamente pode-se escolher qualquer fonte, mas na verdade ela só aparecerá se o computador já tiver a fonte instalada. Portanto, o melhor a se fazer é escolher uma fonte que já venha com o sistema operacional presentes na maioria dos computadores.Isso não quer dizer que você não possa usar mais é recomendado que deixe no modo padrão, mas a exemplos de fontes que você pode usar: Fonte Arial, Fonte Calibri, Fonte Comics Seans MS, Fonte Verdana, etc. Hoje em dia para mudar a fonte do seu código é presciso usar o css que é uma linguagem de estilização, simplificando o seu signficado ela deixa o seu site mais bonito adicionando cores de fundo por exemplo.

Sintaxe: <'spam style="font-size:;">


Exemplo:

<'p>texto da página <'spam style="font-size:20px;">web

texto da página web


Alinhamento do texto

Para destacar o texto, além de efeitos, pode-se alterar a linha ou o parágrafo. Os alinhamentos disponíveis são: Para a direita( padrão), esquerda, centralizado e justificado.

Sintaxe:<'p ALIGN=""><'/p>


Exemplo:

<'p ALIGN="left>texto da página web

texto da página web


Inserindo linhas horizontais:

Você usará essas linhas para dividir a informação em diferentes blocos.

Sintaxe:<'hr>


Exemplo:

<'p>texto da página <'hr>web

texto da página web


Criando subtítulos:

Para isso usaremos a tag sendo que "x" é o número que representa o nível do tópico.

Sintaxe:<'hx><'/hx>


Exemplo:

<'h1>texto da página web<'/h1>

texto da página web
