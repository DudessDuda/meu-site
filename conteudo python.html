Sintaxe

Antes de começar a estudar de verdade, preciso esclarecer uma coisa à você. Antes de você aprender qualquer coisa pratica, você primeiro precisa de uma base teórica. Na programação não é diferente, antes de aplicar um comando, você precisa primeiro saber a sintaxe dele. Comando: É uma forma representativa do que você vai pedir para o computador fazer. Sintaxe: É a estrutura do programa. Agora que você tem essa base, posso apresentar os comandos a você. O comando mais básico e o primeiro que ensinarei a você serve para mostrar uma palavra, frase, texto e outras coisas que veremos mais para frente no curso. O comando em específico é o print.


A sintaxe desse comando é:


print ('x')


(O "X" apresentado nessa sintaxe seria o assunto apresentado por você.)

Exemplo:


print ('Olá mundo')


Quando você tentar executar o este código, aparecerá apenas o "Olá mundo", pois o print apenas mostra o que está dentro dele, então ele não precisa aparecer. No geral, nada aparecerá no seu programa se não existir um print no código, ele é essêncial para que ele funcione. Agora vá praticar e colocar o que você aprendeu aqui em prática, se você não fizer isso, não aprenderá nada.

Variaveis


Agora, sem muita enrolação, vamos aprender do que se trata as variáveis. As variáveis são como pequenas caixas de armazenamento que são "guardadas" dentro da memória do seu computador.


A sintaxe é:

a=x


(Nesse caso, a letra "a" representa a variável que foi criada, já o "X" representa o conteúdo que foi armazenado dentro da variável.)


Exemplo:


nota=9


Agora, você tem uma variável com um valor embutido, mas como foi dito anteriormente, nada aparecerá no programa se não haver um print.

(Antes de continuar, é importante avisar para você não declarar uma variável com números, letras maiúsculas e sinais.)

Então para você visualizar o número que está dentro da variável, você precisará escrever o seguinte comando:


print=nota


Note que você não precisa escrever o 9, pois colocando apenas o nome da variável, será mostrado todo conteúdo embutido nela. Também não é necessário usar as aspas simples, pois elas servem apenas para escrita de textos, aquilo que não for um texto, não se coloca dentro aspas, caso você faça isso, não funcionará, pois a variável, ou qualquer outra coisa que não seja um texto, será apresentado como um texto. Se você quiser apresentar um texto onde no meio dele esteja um valor de uma variável que você criou, você pode usar a seguinte sintaxe:


a=x


print=('y'a)


a = variável

x = valor da variável

y = texto


(Note que o "y" que é o texto, está entre aspas simples a variável está fora, e entre eles, há uma vírgula. O print consegue mostrar os valores entre as aspas e fora, mas a vírgula é necessária para que o programa não dê erro.)


Exemplo:


nota=9


E então, no seu programa aparecerá "Sua nota é 9". Porém, esse é muito complicado de fazer, então mostrarei uma forma mais fácil de colocar uma variável dentro de um print.

A sintaxe é:


nota=5


print=(f'texto{nota}')


Neste comando, o "f" serve para o computador entender que entender que dentro das chaves, irá ser colocado uma variável, sem esse "f" o print não mostrará o conteúdo da variável, e sim as chaves com o nome da variável dentro.


Exemplo:


nota=(f 8)


print()('f sua nota é {nota})


Esse comando também é muito útil caso você queira colocar uma variável entre o texto.

Exemplo:


din =25

print(f'Você tem {din} reais')

Dessa forma, seus programas ficam mais flexíveis e mais rápido de escrever.


Outro comando que é bastante útil é o input. O input trabalha junto com a variável, mas funciona como um print, pois ele permite você implementar um texto para que você consiga "pedir" um valor para a sua variável, assim você pode armazenar o que você quiser, e a variável não precisaria ter um valor fixo. .


Exemplo:


din=input('Qual o valor que tem na sua conta: ')


print(f'Você tem {din} reais')

Esse texto que está embutido na variável, não é um valor atribuído à ela, mas sim uma forma de fazer você guardar o que você quiser guardar nela, por isso geralmente, se usa um texto pedindo ou perguntando algo, e o que você responder, será guardado na variável.

Classificação de variáveis

A classificação de variável vai dizer que tipo de dados a variaveilvai receber, se vai ser número inteiro, número decimal ou string (texto). No python há mais tipos de dados, mas como este curso é básico, aprenderemos apenas esses três tipos. Os três tipos de classificação dos dados são int, str e float. A sintaxe é: a = Classificação(input('Texto'))


int: O tipo de classificação int recebe os números inteiros.

Exemplo:


num=int(input('Digite um número: ') )

Essa é a forma que você usa as classificações, sempre antes do input (vale lembrar que você terá que abrir um novo parêntese também)

str: O tipo de classificação str recebe textos.

Exemplo:


a=str(input('('Digite seu nome: ') )

float: O tipo de classificação floaté usado para receber números decimais (mas também recebe inteiro).

Condições


A última coisa que precisamos aprender neste curso básico são as condições. As condições permitem que você crie dois caminhos para algum resultado que você obtiver com seu programa.

Exemplo:


Você descobre a média de um aluno, se a média for igual ou maior que 6, ele passa de ano, mas se ele tirar menos que isso, ele fica de recuperação. Para usarmos essas condições, usaremos o if elif e else.


Para usarmos essas funções é bem simples. Aqui vai a sintaxe:

if(1° resultado):

Primeira condição


elif(2° resultado):

Segunda condição


else: Mensagem caso não seja nenhuma das condições

Exemplo:


nt1=int(input('('Digite sua primeira nota:') )

nt2=int(input('('Digite sua segunda nota:') )

nt3=int(input('('Digite sua terceira nota:') )

nt4=int(input('('Digite sua quarta nota:') )

media=((nt1+nt2+nt3+nt4) / 4 )

if

media >=6:

print(f'Parabéns sua média é {media} e você passou.' )

elif= 6:

print(f'Vocé não atingiu a média nescessária.' )

else:

print(Você não colocou números, responda as perguntas novamente.')

Você pode usar as condições pra várias situações, isso foi apenas um exemplo, mas se você for criativo, você consegue usar as condições para situações diversas.
